# **App Name**: Career Compass

## Core Features:

- Goal Definition: Allows users to define their desired career goals.
- Skill Pathway Exploration: Suggests optimal skill pathways based on the selected career goals. Will use the LLM as a tool to decide when it is appropriate to incorporate similar careers in its decision-making.
- Portfolio Project Suggestions: Suggests relevant portfolio projects to enhance skills and employability.
- Personalized Career Progression Visualization: Visualizes a personalized career progression based on the user's input and suggested pathways.
- Skill Assessment: Evaluates current skills and provides insights on areas for improvement.
- Job Opportunity Matching: Suggests relevant job opportunities based on the user's skills and career goals.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to evoke trust and professionalism.
- Background color: Light blue (#E8EAF6), nearly white, for a clean, open feel.
- Accent color: Indigo (#5C6BC0), a slightly lighter analogous hue to the primary, to guide user attention.
- Body and headline font: 'Inter', a grotesque-style sans-serif for a modern and objective feel.
- Use simple, clear icons to represent different career paths and skills.
- Maintain a clean and organized layout for easy navigation.
- Use subtle transitions and animations to enhance user experience.
